Parameters
CovariateAlgorithmType: the covariate algorithm type to use for subsequent covariate calculations
TimeMinus1: one minus the current time

BiomassAtTime: biomass at the currrent time

CatchabilityCovariate: the catchability covariate values
Returns
Returns the harvest value evaluated with the appropriate function map algorithm
