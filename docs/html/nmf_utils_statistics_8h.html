◆ LinReg()

void nmfUtilsStatistics::LinReg ( boost::numeric::ublas::matrix< double > &  dataIn,
int &  numColumns,
LinRegOut &  dataOut 
)

This is a public procedure to calculate linear regression coefficients and significance tests. Results are output to a publicly declared variable type LinRegOut that holds the outputs. This should be declared both as a type and public variable in the declarations section of the users code. This procedure accepts an Nx2 matrix of double data containing X and Y values and the Number of data points, N. Any transformations for linearity should be done prior to submitting the data to the procedure.

Parameters
dataIn: matrix of data
numColumns: number of columns in matrix (aka. N); should be > 2
dataOut: output linear regression coefficients
