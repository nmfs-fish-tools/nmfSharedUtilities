Detailed Description

Allows user to interact with MySQL database.

This class allows the user to interact with the MySQL database stored locally on the user's machine. The user can create or delete databases or tables and read from or write to specific database tables. Before using this class the user must assure that MySQL is installed and properly working on the host machine.
