Detailed Description

Definition for the Logger dock widget.

Date
Dec 8, 2016

This file contains the definition for the Logger dock widget. The Logger is capable of displaying log messages generated by the application. Log messages are kept in data files. The Logger can also be used to look at an MSSPM log from a previous run.
